# 🏛️ Municipal Reporting System
### PSD Project 2025-2026 | Track 3

---

## 📋 Project Overview
A robust, high-performance C solution designed for municipal issue tracking. This system provides a centralized platform for citizens to report urban problems, allowing administrative authorities to manage, categorize, and prioritize interventions efficiently.

---

## 🛠️ Technical Architecture
Developed with a focus on **memory optimization** and **algorithmic efficiency**, the project implements:

* **Multi-Index Data Strategy:** Utilizing an Abstract Data Type (ADT) approach with Binary Search Trees (BST) for logarithmic search performance.
* **Zero-Redundancy Memory Management:** Managing records via pointer referencing to ensure minimal memory footprint.
* **Data Persistence:** Full filesystem integration for state preservation across sessions.
* **Standard Compliance:** Strict adherence to **C99** standards with a hand-crafted **Makefile** build system.
